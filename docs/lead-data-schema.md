# סכמת לידים ואבחונים אחידה — GoFlow

כל דף נחיתה ושאלון חדש ישלח את אותה מעטפת נתונים. כך אפשר לנתח בגיליון לפי מקור, עמוד, שאלון, תוצאה וקמפיין — בלי לבנות גיליון חדש לכל דף.

## מזהה ורישום

| עמודה מוצעת בגיליון | שדה |
|---|---|
| תאריך ושעה | `submitted_at` |
| מזהה מילוי | `quiz_attempt_id` |
| גרסת סכמה | `schema_version` |
| סוג ליד | `lead_type` |
| מקור הטופס | `source` |

## כלל ל־Apps Script

הסקריפט צריך למצוא את מספר העמודה לפי שם הכותרת בשורה הראשונה, ולא לפי מיקום קבוע. שדה לא מוכר יתווסף רק אם קיימת כותרת תואמת. כך ניתן להוסיף בעתיד עמוד או שאלון בלי לשבור נתונים קיימים.
